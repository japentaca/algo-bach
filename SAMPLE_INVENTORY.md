## Key Findings & Patterns

### 1. **Naming Convention Inconsistencies**
- **Lowercase Pattern:** Most instruments use lowercase note names with SHARP/FLAT modifiers
  - Example: `violin-aSHARP3.mp3`, `flute-dSHARP4.mp3`
- **Uppercase Pattern:** Some (Trombone, Tuba, Trumpet section) use uppercase
  - Example: `A2.mp3`, `CSHARP3.mp3`
- **Prefix Variations:** Ensemble versions often use different prefixes than solos
  - Solo: `horn-`, Ensemble: `horns-stc-rr1-`

### 2. **Articulation Variants**
- **Pizzicato (piz):** String instruments with 2 round-robin versions (rr1, rr2)
- **Staccato (stc):** Brass/Wind ensembles with 2 round-robin versions (rr1, rr2)
- **Sustained (sus):** Most instruments have a sustained version
- **Dynamic Levels:** Piano has separate Forte (f) and Piano (p) samples

### 3. **Note Coverage Gaps**
- **String Instruments:** Strong coverage at A, C, D#, F# (pentatonic-like pattern)
- **Woodwinds:** Good coverage but with gaps, especially for extended ranges
- **Brass:** More complete coverage with multiple articulation variants
- **Piano:** Extremely comprehensive, full chromatic range with dynamics
- **Percussion:** Minimal note count but full dynamic range

### 4. **Octave Ranges**
- **Wide Range (8+ octaves):** Piano (1-8)
- **Soprano (5-7 octaves):** Violin, Flute, Trumpet, Piccolo
- **Alto (4-6 octaves):** Oboe, Clarinet, Alto Flute, Violas
- **Tenor (3-5 octaves):** Horn, Trombone, Cello
- **Bass (1-4 octaves):** Tuba, Bass Clarinet, Basses, Contrabassoon

## Format for instrumentConfig.js Updates

Each instrument should include:

```javascript
'InstrumentName': {
  baseUrl: 'Orchestra/FolderName/',
  samples: {
    'Note1': 'prefix-note-octave.mp3',
    'Note2': 'prefix-note-octave.mp3',
    // ... complete note mapping
  },
  // Optional: articulations, dynamics, etc.
  articulations: ['solo', 'staccato_rr1', 'staccato_rr2', 'sustained'],
  dynamics: ['forte', 'piano']
}
```

### Tonal Library Note Format
Use format: `C4`, `C#4`, `Db4` (standard Tonal.js notation)
- No lowercase
- Sharp: `#` (not `SHARP`)
- Flat: `b` (not `FLAT`)
- Octave: 0-8
